---
title: "Get Comments"
api: "GET /bookmarks/:id/comments"
description: "Retrieve paginated comments for a specific bookmark"
---

## Path Parameters

<ParamField path="id" type="string" required>
  The public ID of the bookmark (format: `bkm_*`)
</ParamField>

## Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="integer" default="20">
  Number of comments per page
</ParamField>

## Response

<ResponseField name="comments" type="array">
  Array of comment objects with owner information
  
  <Expandable title="Comment Object">
    <ResponseField name="id" type="string">
      The comment's public ID (format: `cmt_*`)
    </ResponseField>
    
    <ResponseField name="user_id" type="string">
      Public ID of the user who created the comment
    </ResponseField>
    
    <ResponseField name="bookmark_id" type="string">
      Public ID of the bookmark this comment belongs to
    </ResponseField>
    
    <ResponseField name="parent_id" type="string" default="null">
      Public ID of the parent comment if this is a reply
    </ResponseField>
    
    <ResponseField name="content" type="string">
      The comment text content
    </ResponseField>
    
    <ResponseField name="status" type="string">
      Comment status: `active` or `hidden`
    </ResponseField>
    
    <ResponseField name="owner_id" type="string">
      Public ID of the comment author
    </ResponseField>
    
    <ResponseField name="owner_name" type="string">
      Display name of the comment author
    </ResponseField>
    
    <ResponseField name="owner_username" type="string">
      Username of the comment author
    </ResponseField>
    
    <ResponseField name="owner_avatar" type="string">
      Avatar URL of the comment author
    </ResponseField>
    
    <ResponseField name="liked" type="boolean">
      Whether the authenticated user has liked this comment
    </ResponseField>
    
    <ResponseField name="likes_count" type="integer">
      Total number of likes on this comment
    </ResponseField>
    
    <ResponseField name="replies_count" type="integer">
      Total number of replies to this comment
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      ISO 8601 timestamp of when the comment was created
    </ResponseField>
    
    <ResponseField name="updated_at" type="string">
      ISO 8601 timestamp of when the comment was last updated
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="pagination" type="object">
  Pagination metadata
  
  <Expandable title="Pagination Object">
    <ResponseField name="page" type="integer">
      Current page number
    </ResponseField>
    
    <ResponseField name="limit" type="integer">
      Number of items per page
    </ResponseField>
    
    <ResponseField name="total" type="integer">
      Total number of comments
    </ResponseField>
    
    <ResponseField name="total_pages" type="integer">
      Total number of pages
    </ResponseField>
  </Expandable>
</ResponseField>

## Authentication

Requires authentication. User must have access to the bookmark to view comments.

## Pagination

Comments are returned in paginated format. Use the `page` and `limit` query parameters to navigate through results. The response includes pagination metadata showing the current page, total items, and total pages.

## Filtering

Only comments with `status: "active"` are returned. Hidden comments are filtered out from the results.

## Comment Metadata

Each comment includes:
- **Owner information**: Name, username, and avatar of the comment author
- **Engagement metrics**: Like count and reply count
- **User interaction**: Whether the authenticated user has liked the comment
- **Hierarchy**: Parent ID for nested replies

## Error Responses

<ResponseField name="400" type="Bad Request">
  Invalid bookmark ID format or pagination parameters
</ResponseField>

<ResponseField name="403" type="Forbidden">
  User doesn't have access to this bookmark
</ResponseField>

<ResponseField name="404" type="Not Found">
  Bookmark not found
</ResponseField>
