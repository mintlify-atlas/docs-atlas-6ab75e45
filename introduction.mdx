<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Get started in minutes
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Follow these steps to set up Cultus and make your first API request
  </p>

  <Steps>
    <Step title="Clone and configure">
      Clone the repository and set up your environment variables:
      
      ```bash
      git clone https://github.com/linkvite/cultus.git
      cd cultus
      cp .env.example .env
      ```
      
      Edit `.env` with your database credentials, JWT secret, and other required configuration.
    </Step>
    
    <Step title="Start the services">
      Launch the API server and required dependencies:
      
      ```bash
      docker-compose -f deployments/docker-compose.dev.yml up -d
      ```
      
      The API will be available at `http://localhost:9000`
    </Step>
    
    <Step title="Create your first user">
      Register a new user account:
      
      ```bash cURL
      curl -X POST http://localhost:9000/auth/signup \
        -H "Content-Type: application/json" \
        -d '{
          "email": "user@example.com",
          "password": "secure_password",
          "username": "myusername"
        }'
      ```
      
      <Accordion title="Example response">
      ```json
      {
        "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
        "refresh_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
        "user": {
          "id": "usr_abc123",
          "email": "user@example.com",
          "username": "myusername"
        }
      }
      ```
      </Accordion>
    </Step>
    
    <Step title="Make your first authenticated request">
      Use the access token to create a bookmark:
      
      ```bash cURL
      curl -X POST http://localhost:9000/bookmarks \
        -H "Content-Type: application/json" \
        -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
        -d '{
          "url": "https://example.com",
          "title": "My First Bookmark",
          "tags": ["example", "test"]
        }'
      ```
      
      You've successfully created your first bookmark! Explore the [API Reference](/api/bookmarks/create) for more endpoints.
    </Step>
  </Steps>
</div>
